Virtualisation de Moodle

Base de donnée d'instance

Ce champ permet de définir l'hôte (hôte:port) où réside la base de données sur laquelle doit fonctionner l'instance. Cet hôte doit avoir une base de données installée et opérationnelle, et doit être accessible à partir du serveur où le module de virtualisation est opéré.
